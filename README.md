# Sharp AI — Build AI Bots. Deploy Everywhere.

Sharp AI is an ultra-premium, futuristic platform for building, deploying, and monetizing AI bots. Built with a "Senior SDE" caliber design, it features an immersive 3D interactive atmosphere and a high-performance visual engine.

---

## ✨ World-Class Aesthetics: Solaris 3D

The platform has been transcended into the **Solaris 3D** aesthetic—a masterpiece of modern web design:

- **Solaris 3D Engine**: A mathematically driven, interactive point-cloud globe background (1,200 points, Fibonacci distribution) built using pure Canvas 3D projection.
- **"Black Ice" Glassmorphism**: High-contrast UI containers with **0.92 opacity** and **42px soft-motion backdrop blurs**, ensuring razor-sharp legibility against the dynamic background.
- **Deep Space Palette**: A sophisticated color system using **Solaris Gold**, **Nova Crimson**, and **Absolute Deep Space Black** (#010103).
- **Interactive 4D Depth**: A responsive visual layer that reacts to mouse movement and scroll, creating an immersive sense of volume.

## 🚀 Core Features

### 🧠 Modern Bot Builder
- **No-Code Interface**: Build complex AI personalities in minutes.
- **Model Agnostic**: Support for GPT-4o, Claude 3.5, Gemini 1.5, Llama 3, and Mixtral.
- **Knowledge Base**: Upload files (.pdf, .txt, .md) or connect URLs to give your AI custom context.
- **Interactive Preview**: Test your bot in a real-time "Black Ice" chat interface before launching.

### 🔌 Multi-Channel Deployment
- **One-Click Connect**: Deploy to **Telegram**, **Discord**, and **Slack**.
- **WebChat Widget**: Embed a luminous, floating chat bubble on any website with a single line of code.

### 🛒 Bot Marketplace
- **Monetize Your Brain**: Submit your proven bot templates to the marketplace.
- **Template Ecosystem**: Install community-built AI templates (Support, Sales, Education, etc.) in one click.

### 💰 Tokenomics ($Sharp)
- **$Sharp Utility**: Earn tokens by contributing templates or performing tasks.
- **Transparent Allocation**: View the on-chain verifiable allocation and vesting schedules directly on the platform.

## 🛠️ Technical Stack

- **Frontend**: Vanilla HTML5, CSS3 (Modern Flex/Grid), and Pure JavaScript.
- **Interactive Layer**: 3D Mathematical Projection Engine (Canvas 2D API).
- **Typography**: Space Grotesk (Titles) & DM Sans (Interface).
- **Backend**: Node.js with a robust API layer for AI orchestration.

---

## 🛡️ Security & Integrity

- **History Purged**: All sensitive environment variables have been scrubbed from the repository history using `git filter-branch`.
- **Exclusion Standard**: Robust `.gitignore` structure to prevent tracking of secrets, logs, and bulky dependencies.

## 🏁 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sidd1104/Sidd-AiDemo.git
   ```
2. **Setup the Backend**:
   - Navigate to `server/`
   - Create a `.env` file (see `.env.example`)
   - `npm install`
   - `npm start`
3. **Launch the Frontend**:
   - Open `index.html` in your browser (or use a live server on port 3001).
